Haar Wavelet Transform Class:


The Haar Wavelet Transform class has the purpose of taking in a signal and outputing the signal processed by the Haar Wavelet Transform. It is dependent on the convolution function for computation. It is also dependent on the haar function generator to establish a Haar Scaling and Wavelet vector. For practical I/O, a result plotter is also necessary.


All of the above can be accommodated within one class. One other class that is necessary is the myVector class. This class provides a simple, yet practical data type for the computation. It can be allocated and deallocated such that its size is appropriate for the task. It is projected that the two-dimensional version will simply add a matrix manipulator. What such a manipulator extracts rows and columns into myVector structures such that the computation to procede on each row and column..
