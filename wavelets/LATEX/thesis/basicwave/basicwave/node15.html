One Dimensional Wavelet Transform

The primary function in the Wavelet transform class is the one dimensional wavelet transform (hWaveXform). More than one form of this function could possible exist; however, for simplisty only one was produced. It takes two myVector classes (input and output). It produces a haar difference and scaling vector, as well as a , , , and myVector classes for use within the function. The algorithm is as follows is the size of S (or the input)                           T = join (,)         F = evenSplit (T)         ForceInsert (F, output)         end = floor ( )         Extract (output, S, 0, end)


The Extract procedure is to take values from output up to the index end, and make S a copy of that vector. Thus S is called the resolution vector. T is always twice the size of the working S vector. F is always half. The resolution vector decreases by half each iteration, and starts at the same size as the input.
